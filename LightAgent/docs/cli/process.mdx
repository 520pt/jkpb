---
title: Process Management
description: Manage LightAgent process lifecycle with lightagent commands
---

Process management commands control the LightAgent background process. These commands are only available in the terminal.

## start

Start the LightAgent service. Runs as a background daemon by default and automatically tails logs.

```bash
lightagent start
```

**Options:**

| Option | Description |
| --- | --- |
| `-f`, `--foreground` | Run in foreground, not as a background daemon |
| `--no-logs` | Don't tail logs after starting |

## stop

Stop the running LightAgent service.

```bash
lightagent stop
```

## restart

Restart the LightAgent service (stop then start).

```bash
lightagent restart
```

**Options:**

| Option | Description |
| --- | --- |
| `--no-logs` | Don't tail logs after restart |

## update

Update code and restart the service. Automatically performs:

1. Pull latest code (`git pull`)
2. Stop current service
3. Update Python dependencies
4. Reinstall CLI
5. Start service

```bash
lightagent update
```

<Warning>
  If `git pull` fails (e.g., uncommitted local changes), the update aborts and the service remains unaffected.
</Warning>

## status

Check LightAgent service status, including process info, version, and current model/channel configuration.

```bash
lightagent status
```

## logs

View service logs.

```bash
lightagent logs
```

**Options:**

| Option | Description | Default |
| --- | --- | --- |
| `-f`, `--follow` | Continuously tail log output | No |
| `-n`, `--lines` | Show last N lines | 50 |

Examples:

```bash
# View last 100 lines
lightagent logs -n 100

# Continuously tail logs
lightagent logs -f
```

## install-browser

Install Playwright and Chromium browser for the [browser tool](/tools/browser).

```bash
lightagent install-browser
```

<Tip>
  Only needed when using browser tools (web browsing, screenshots, etc.).
</Tip>

## run.sh Compatibility

If LightAgent CLI is not installed, you can use `run.sh` to manage the service:

| lightagent command | run.sh equivalent |
| --- | --- |
| `lightagent start` | `./run.sh start` |
| `lightagent stop` | `./run.sh stop` |
| `lightagent restart` | `./run.sh restart` |
| `lightagent update` | `./run.sh update` |
| `lightagent status` | `./run.sh status` |
| `lightagent logs` | `./run.sh logs` |

<Note>
  The `lightagent` command is recommended — it provides cleaner syntax and richer features. It is automatically installed via the one-click install script.
</Note>
